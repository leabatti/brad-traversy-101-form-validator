# Form Validator — Brad Traversy 50 Projects

Basic form validation using HTML, CSS, and JavaScript.  
This project replicates the exercise from Brad Traversy’s *50 Projects in 50 Days* course, with the goal of practicing simple frontend validation logic.

## Features
- Required field checks  
- Minimum and maximum length validation  
- Email validation using a regular expression  
- Password confirmation  
- Visual feedback for errors and success states  

## Technologies Used
- HTML5  
- CSS3  
- Vanilla JavaScript  

## Project Purpose
Practice basic form validation and reinforce DOM manipulation concepts.

## How to Use
1. Open `index.html` in your browser.  
2. Fill out the form to see the validation in action.
